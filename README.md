Video Chatting Website
A real-time video chat application built with React and powered by ZegoCloud for seamless video and audio communication.

🚀 Features
✅ One-on-One and Group Video Calls
✅ High-Quality Audio & Video Streaming
✅ Screen Sharing
✅ Chat Messaging During Calls
✅ User Authentication
✅ Secure & Scalable Infrastructure

🛠 Tech Stack
Frontend: React, Tailwind CSS
Video SDK: ZegoCloud
Backend: Node.js, Express.js (optional)
Authentication: Firebase/Auth0 (optional)
🔧 Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/YourGitHubUsername/VideoChatApp.git
cd VideoChatApp

2️⃣ Install Dependencies
npm install

3️⃣ Configure ZegoCloud SDK
Sign up on ZegoCloud
Get your App ID and App Sign
Create a .env file and add:
env

REACT_APP_ZEGO_APP_ID=your_app_id
REACT_APP_ZEGO_APP_SIGN=your_app_sign
4️⃣ Run the App
npm start
